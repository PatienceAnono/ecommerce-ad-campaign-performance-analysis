# 📊 Project Overview
This project analyzes the performance of multiple**e-commerce advertising campaigns** across different marketing platforms including:
- Google Ads
- Meta Ads
- TikTok Ads
- Email Marketing
- Affiliate Marketing
The goal of the analysis was to evaluate campaign effectiveness, identify high-performing channels, measure marketing efficiency and generate actionable insights to support data-driven marketing decisions. Using Excel, the dataset was cleaned, key marketing metrics were calculated, pivot tables were built and an interactive dashboard was developed to present campaign performance clearly to stakeholders.

 ---
 
# 🗂 Dataset Description The dataset contains advertising campaign performance data with the following fields: | Column | Description | |------|-------------| | Campaign_ID | Unique identifier for each campaign | | Campaign_Name | Campaign marketing name | | Platform | Advertising channel | | Date | Campaign date | | Impressions | Number of ad views | | Clicks | Number of clicks | | Conversions | Number of purchases | | Ad_Spend | Total advertising cost | | Revenue | Revenue generated | | CTR | Click Through Rate | | Conversion_Rate | Conversion efficiency | | CPA | Cost Per Acquisition | | ROAS | Return on Ad Spend | | Year | Campaign year | | Month | Campaign month | | Month_Number | Numeric month value | 

--- 

# 📈 Key Marketing Metrics Several important marketing performance metrics were calculated: 
### CTR (Click-Through Rate) ``` CTR = Clicks / Impressions 

``` 
### Conversion Rate

``` Conversion Rate = Conversions / Clicks

``` ### CPA (Cost Per Acquisition) ``` CPA = Ad Spend / Conversions ``` ### ROAS (Return on Ad Spend) ``` ROAS = Revenue / Ad Spend ``` These metrics help evaluate **marketing efficiency, campaign profitability, and customer engagement**. --- # 📊 Dashboard Preview ![Dashboard Preview](dashboard_preview.png) The dashboard summarizes campaign performance through: - KPI performance cards - Platform revenue comparison - ROAS by platform - Monthly revenue trend - Revenue contribution by campaign --- # ⭐ Key Insights Key insights identified from the analysis include: • **Google Ads generated the highest revenue**, making it the most impactful marketing channel. • **Email Marketing produced the strongest ROAS**, indicating the most efficient use of advertising spend. • **Meta Ads delivered consistent revenue performance** across campaigns. • **TikTok Ads showed lower ROAS**, suggesting potential opportunities for campaign optimization. • Promotional campaigns such as **Holiday Deals and Flash Sales** performed particularly well. --- # 💡 Strategic Recommendations Based on the findings, several recommendations were identified: - Increase investment in **high-ROAS channels such as Email Marketing** - Optimize lower-performing channels such as **TikTok Ads** - Leverage **seasonal campaigns and promotions** during peak periods - Continue monitoring **CTR, Conversion Rate, CPA, and ROAS** to optimize marketing performance --- # 📁 Repository Contents This repository contains the following files: ``` ecommerce_ad_campaign_dashboard.xlsx ecommerce_ad_campaign_dataset.csv ecommerce_ad_campaign_report.pdf dashboard_preview.png README.md ``` --- # 🎯 Business Value This project demonstrates how marketing campaign data can be transformed into **actionable business insights** through structured analysis and dashboard reporting.
